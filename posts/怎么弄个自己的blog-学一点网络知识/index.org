本来准备在介绍完数据库后介绍 PHP，然后结束掉 LAMP 的搭建，但在此之前还是先学一些网络知识，不然 PHP 感觉没法讲了。不过话又说回来网络这一部分我本来准备在介绍 HTTP 服务器时顺便说一下的，但发现内容太多了，正好都放在这篇文章里。

就写博客应用这个目的来说，我们只需要学学 HTTP 怎么弄就行了，应用层以下的东西完全不用管。但我这《计算机网络：自顶向下方法》也不能白看，所以本文会对计算机网络做一个简短但是全面的介绍，这应该能代表目前我对网络的最高水平的认识了（有机会的再继续了解吧）。在完成这些后再说说我们能在应用层上做什么。同时，计算机网络的层次也是一个很不错的抽象例子，借此机会也聊聊我对抽象的认识。

本文的目的是写一篇“哪怕是很多年之后我完全忘了网络也能看懂”的文章，希望如果你完全没有了解过网络知识也能比较容易地通过本文学到一些基础。部分或完全看不懂也没事，在网络上总能找到适合不同人的教程。

本文使用的环境如下：

- Ubuntu Server 22.04.1 LTS on VMware 16.1.2 on Windows 10
